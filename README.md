# DishSwipe 🍽️

## School Project

DishSwipe is a school project built as part of a front-end development assignment using **HTML, CSS, and Vanilla JavaScript**. The goal of the project is to build a responsive and interactive website that connects to an external API and displays real-world data in a dynamic way.

The application allows users to explore recipes using two main features:
- A search system for finding specific recipes
- A “Smash or Pass” mode where random recipes are shown one at a time for quick interaction

If a user selects a recipe they like, they can open a detailed view that shows ingredients and cooking instructions. If they skip it, a new random recipe is shown.

The project uses the **TheMealDB API** to fetch recipe data and demonstrates working with APIs, DOM manipulation, and responsive design.

This project was created for educational purposes to practice:
- HTML structure and semantic layout
- CSS responsive design
- JavaScript logic and interactivity
- Working with external APIs
